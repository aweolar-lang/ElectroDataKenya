**A real-time, data-driven voting Prediction platform for the Kenyan 2027 General Election.**

StudyLite Kenya is a modern web application designed to empower citizens with transparent, real-time data insights. It simulates the voting process for President, Governors, and Members of Parliament (MPs), providing instant visualization of results and trends across Counties and Constituencies.

## üöÄ Key Features

* **Real-Time Voting Simulation:** Users can cast votes for Presidential, Gubernatorial, and Parliamentary candidates.
* **Live Tallying:** Results update instantly without page reloads using optimistic UI and polling.
* **Dynamic Constituency Logic:** Smart filtering of MP candidates based on the selected constituency.
* **Interactive Maps & Graphs:** Visual representation of data using dynamic bar charts and county-level breakdowns.
* **Session Security:** Uses anonymous browser fingerprinting (`anon_session`) to ensure data integrity and prevent spamming.
* **GDPR/Privacy Compliant:** Includes a consent guard to ensure users agree to participate responsibly.
